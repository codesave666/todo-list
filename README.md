# 纯前端待办清单工具

一个简洁美观的纯前端待办清单工具，单HTML文件即可使用，支持任务卡片、拖拽排序和分类管理。

## 功能特点

### 🚀 即开即用
- 单HTML文件，无需安装任何软件
- 双击HTML文件即可在浏览器中打开使用（file://协议）
- 数据保存在本地浏览器中，刷新页面不会丢失

### 📝 任务管理
- 添加、编辑、删除任务
- 任务卡片式展示，清晰美观
- 支持任务标题、描述和分类
- 显示任务创建时间

### 🗂️ 分类管理
- 预置工作、个人、学习、购物等分类
- 支持添加自定义分类
- 按分类筛选任务
- 每个分类有专属图标

### 🎯 拖拽排序
- 通过拖拽任务卡片重新排序
- 直观的拖拽视觉反馈
- 排序结果自动保存

### 📱 响应式设计
- 适配桌面端和移动端
- 在不同屏幕尺寸下都有良好的显示效果

## 使用方法

1. 下载 `index.html` 文件到本地
2. 双击 `index.html` 文件在浏览器中打开
3. 开始添加和管理你的待办任务

### 添加任务
1. 在"添加新任务"表单中填写任务标题
2. 选择或创建分类
3. （可选）填写任务描述
4. 点击"添加任务"按钮

### 管理分类
- 左侧侧边栏显示所有分类
- 点击分类按钮筛选该分类下的任务
- 点击"全部任务"查看所有任务
- 在"添加新分类"输入框中输入新分类名称，点击"添加"按钮

### 编辑任务
- 点击任务卡片上的"编辑"按钮（铅笔图标）
- 表单会自动填充当前任务信息
- 修改后点击"更新任务"按钮保存

### 删除任务
- 点击任务卡片上的"删除"按钮（垃圾桶图标）
- 确认后任务将被删除

### 拖拽排序
- 将鼠标悬停在任务卡片上，光标变为移动图标
- 按住并拖动任务卡片到新位置
- 松开鼠标完成排序

## 技术特点

- **纯前端实现**：仅使用HTML、CSS和JavaScript
- **本地存储**：使用LocalStorage保存数据
- **原生拖拽API**：使用HTML5原生拖放功能实现排序
- **响应式布局**：使用Flexbox和CSS Grid实现
- **无外部依赖**：所有代码集成在单个HTML文件中

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
- 支持file://协议本地运行
- 不需要网络连接（离线可用）

## 数据安全

- 所有数据仅保存在用户本地浏览器中
- 不会上传到任何服务器
- 清除浏览器缓存会删除所有数据

## 注意事项

1. 如果需要在不同设备间同步数据，请手动导出/导入数据
2. 清除浏览器缓存会导致数据丢失
3. 建议定期备份重要任务

## 自定义和扩展

- 修改CSS样式来自定义外观
- 调整JavaScript逻辑来改变功能
- 添加新的分类图标到 `getCategoryIcon` 函数
